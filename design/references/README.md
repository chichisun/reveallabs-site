# Landing Page References

Persistent reference materials for the reveal. landing page v1 build.

All paths in `LANDING-PAGE-GAMEPLAN.md` should resolve here, not to `/tmp/` (which is ephemeral).

## Contents

### `viktor-oddy/`
Viktor Oddy's Claude Design workflow for building animated, award-winning websites. Downloaded from X (April 18 2026), transcribed, frames sampled.
- `transcript.txt` — full 18-min tutorial transcript
- `WORKFLOW.md` — distilled section-by-section pattern I extracted
- `frames-sample/` — representative keyframes showing Claude Design UI at different workflow stages

### `mojang/`
Mojang Studios logo animation — the visual reference Kase chose for the brand entrance (rotation → reveal pattern).
- `mojang-studios-animation.mp4` — 4.4s source video (226KB)
- `frames/` — 17 extracted keyframes
- `ANALYSIS.md` — what the animation actually does (the rotation trick, not morphing)

### `claude-design-guide.md`
How Claude Design works, credit management tips, export format, known limitations. Pulled from Anthropic's launch announcement (April 17 2026) and early user reports.

## When to open these

- **Viktor workflow** → open before Track F (Claude Design build). The section-by-section pattern shapes how to prompt.
- **Mojang analysis** → open before Track F-11 (brand entrance animation). Reference for how rotation resolves into wordmark.
- **Claude Design guide** → open at start of Track F. Understand credit limits before burning through them.
